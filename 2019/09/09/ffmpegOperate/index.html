参数详解

参数 描述
-i 输入您要处理的视频文件路径
视频地址 可以是视频文件，也可以是视频流
-vcodec copy 表示使用跟原视频一样的视频编解码器
-acodec copy 表示使用跟原视频一样的音频编解码器
-ss 开始时间 从某个时间开始
-to 结束时间 从某个时间结束
-y 表示如果输出文件已存在则覆盖
