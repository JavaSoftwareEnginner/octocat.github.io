树莓派开发板Ubuntu系统及桌面安装

镜像下载

进入ubuntu-mate官网下载镜像

特点：镜像包较大，自带桌面，自带raspi-config

进入ubuntu官网下载镜像

特点：镜像包很小，无桌面，无raspi-config（目前版本暂无）

找到Ubuntu for IoT下载
笔者下载的文件名为ubuntu-18.04.3-preinstalled-server-armhf+raspi2.img.xz
树莓派2或3系统版本也不相同

现在安装工具balenaEtcher

ssh默认账号密码都是“ubuntu”，切换 root账户：

sudo -i

安装Windows远程连接

apt-get install xrdp

如果安装出错，部分源出现404，源修改为raspi官方源如下

sudo vim /etc/apt/sources.list

在第一行添加

deb http://mirrordirector.raspbian.org/raspbian/ stretch main contrib non-free rpi

保存，更新源

apt-get update

安装可视化桌面

apt-get install xfce4

重启树莓派

sudo init 6

至此安装完毕

卸载程序

sudo apt-get remove ***

sudo apt-get purge ***
